# UI 重构说明

## 概览
根据提供的UI设计图，对整个网站进行了完整的重构，采用简洁、现代的设计风格。

## 主要变更

### 1. 首页 (`/`)
- **布局**：两栏大型分类卡片布局
- **分类**：
  - People (人物摄影)
  - Interiors (室内设计)
- **交互效果**：
  - 鼠标悬停时图片放大
  - 底部边框指示器
  - 叠加层效果

### 2. 导航栏 (Header)
- **设计**：简洁的固定顶部导航
- **左侧**：网站标题 "Your Site Title"
- **右侧**：导航链接
  - Work (作品集)
  - Contact (联系方式)
  - Login (登录)
- **样式**：纯白背景，黑色文字，悬停透明度效果

### 3. 新增页面

#### Work 页面 (`/work`)
- 展示所有作品项目
- 2列网格布局
- 包含项目标题、分类和年份

#### Contact 页面 (`/contact`)
- 完整的联系表单
- 包含姓名、邮箱、主题、消息字段
- 额外联系信息（邮箱、电话、地址）

#### Login 页面 (`/login`)
- 简洁的登录表单
- 邮箱和密码输入
- 记住我选项和忘记密码链接

#### People 分类页面 (`/category/people`)
- 人物摄影作品集
- 3列网格布局
- 6个作品展示

#### Interiors 分类页面 (`/category/interiors`)
- 室内设计作品集
- 3列网格布局
- 6个作品展示

#### About 页面 (`/about`)
- 更新为简洁风格
- 左侧：个人照片和数据统计
- 右侧：个人介绍和专业领域
- CTA按钮链接到联系页面

### 4. Footer (页脚)
- 简化设计
- 版权信息
- 隐私政策和服务条款链接

### 5. 全局样式更新
- 添加了自定义滚动条样式
- 优化了字体渲染
- 添加了平滑滚动效果

### 6. Tailwind 配置
创建了 `tailwind.config.ts`：
- 自定义 aspect ratio 值
- 字体配置
- 扩展主题设置

## 技术栈
- Next.js 16
- React 19
- TypeScript
- Tailwind CSS 4
- Unsplash API (图片资源)

## 设计特点
1. **极简主义**：干净的布局，大量留白
2. **黑白配色**：主要使用黑色和白色，灰色作为辅助
3. **大胆的图片展示**：图片占据主导地位
4. **微妙的交互**：平滑的过渡和悬停效果
5. **响应式设计**：适配桌面和移动设备

## 运行项目
```bash
npm run dev
```

访问 http://localhost:3000 查看效果

## 页面路由
- `/` - 首页（分类导航）
- `/work` - 作品集列表
- `/about` - 关于页面
- `/contact` - 联系页面
- `/login` - 登录页面
- `/category/people` - 人物摄影作品
- `/category/interiors` - 室内设计作品
